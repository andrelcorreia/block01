Challenges:

01 => First, you'll be given an input of array of numbers, you'll need to divide them up and list them.

For example:

value = [387421]

return = [1,2,3,4,7,8]

If the number repeats, use it only once.

For example:

value = [3862301]

return = [0,1,2,3,6,8]
